# Dormitory Management System

ระบบจัดการหอพัก (System Dormitory)

## Features
- ระบบ Login (Admin / Tenant)
- จัดการห้องพัก
- ระบบแจ้งซ่อม
- ระบบออกใบแจ้งหนี้

## Tech Stack
- Backend: Node.js + Express
- Frontend: React
- Database: MySQL
- Auth: JWT
